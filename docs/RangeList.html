Class: RangeList

RangeList()

Class to create a range list object. A pair of integers define a range. For example: [1, 5). This range includes integers: 1, 2, 3, and 4. A range list is an aggregate of these ranges: [1, 5), [10, 11), [100, 201). See RangeList Examples.
